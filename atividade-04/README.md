# 🎟️ Ticket de Saída

### 1. Qual trilha seu grupo escolheu?
Escolhemos a trilha de **Sistemas de Informação Geográfica (SIG) / Mapeamento Territorial e Impacto Social**.

### 2. Em uma frase, qual o problema real identificado?
A falta de centralização e acessibilidade de dados geoespaciais e jurídicos dificulta o monitoramento do status de regularização das terras quilombolas e atrasa a implementação de políticas públicas.

### 3. Como um sistema pode ajudar nesse problema?
O sistema pode unificar as bases de dados do INCRA, IBGE e Fundação Palmares em um mapa interativo e visual, permitindo acompanhar o progresso das titulações e identificar gargalos burocráticos em tempo real.

### 4. Qual a diferença entre sistema e software para você?
O **software** é a aplicação cartográfica ou a plataforma web construída (o código e as ferramentas de mapa), enquanto o **sistema** é o todo: o software rodando, os servidores hospedando os dados geográficos, os analistas alimentando as informações e as comunidades quilombolas utilizando esses dados para defender seus direitos.
